Продолжаем работать над проектом FraudDetector.
Если вы его еще не делали, рекомендуем для начала пройти именно его.

Напомним легенду проекта.

Представьте, что вы работаете разработчиком в трейдинговой компании.

В трейдинговой компании каждый день трейдеры совершают большое количество транзакций.

Каждая транзакция в системе должна быть проверена на мошенничество (fraud).

Вам поручено создать систему автоматического обнаружения мошенничества (Fraud Detector).

Ваша система должна проверять транзакции на основе правил, которые вам будут рассказаны чуть позже.

Сеньор разработчик уже создал для вас несколько классов в коде и тесты. Пожалуйста, ознакомьтесь с ними!
