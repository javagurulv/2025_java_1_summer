Краткое описание проекта Traders and Transactions.

Проект Traders and Transactions является хорошим введением в
функциональное программирование и Stream API.

Заданий: 13
Сложность: легкий
Для начинающих.

Цель проекта: на практическом примере домена трейдеров и транзакций произвести
погружение и разобрать основные принципы работы Stream API в Java. Stream API
это не заменимый инструмент любого разработчика на Java для обработки данных.
В реальных проектах уже давно никто не пишет циклы, для обработки данных
и не только используют функциональную парадигму программирования и Stream API.
Это удобно, это экономит время, это минимизирует количество строк кода,
делает код чище и понятнее.
